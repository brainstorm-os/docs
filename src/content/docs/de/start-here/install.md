Brainstorm ist eine Desktop-Anwendung für **macOS, Windows und Linux**.

:::note[Beta]
Brainstorm ist in offener Beta. Hol dir den Build für deine Plattform von der [Downloads-Seite](https://getbrainstorm.online/de/downloads) — jede Version wird dort veröffentlicht.
:::

## Systemanforderungen

- **macOS** 12 (Monterey) oder neuer — Apple Silicon und Intel.
- **Windows** 10 oder neuer (64-Bit).
- **Linux** — eine aktuelle 64-Bit-Distribution (AppImage / `.deb`).
- ~400 MB Speicherplatz für die App, plus das, was deine Vaults fassen.

## Installieren

Lade den Build für deine Plattform von [getbrainstorm.online/de/downloads](https://getbrainstorm.online/de/downloads) herunter, dann:

1. **macOS** — öffne die `.dmg` und ziehe Brainstorm in den Programme-Ordner. Beim ersten Start: Rechtsklick → Öffnen, um Gatekeeper zu bestätigen.
2. **Windows** — führe das Installationsprogramm aus und folge den Anweisungen.
3. **Linux** — mach das AppImage ausführbar (`chmod +x`) und starte es, oder installiere die `.deb` mit deinem Paketmanager.

## Erster Start

Beim ersten Start bittet dich Brainstorm, einen **Vault zu erstellen** — das Zuhause deines Wissens auf der Festplatte. Wähle einen Ordner, entscheide, wie du ihn schützt (System-Schlüsselbund oder eine Passphrase), und du bist drin. Die vollständige Anleitung steht im [Schnellstart](/de/start-here/quickstart/).

## Aktualisieren

Apps innerhalb von Brainstorm aktualisieren sich unabhängig von der Shell. Die Shell selbst prüft beim Start auf Updates und wendet sie im Hintergrund an; deine Vaults werden von einem Update nie berührt.
