**Kalender** ist eine zeitbasierte Linse über deinem Vault. Alles mit einem Datum — ein Termin, das Fälligkeitsdatum einer Aufgabe, die Frist einer Notiz — kann darauf erscheinen.

## Was es kann

- Wiederkehrende Termine auf einer vollständigen RRULE-Engine, bearbeitet über einen Wiederholungs-Editor in einfacher Sprache.
- ICS-Import und -Export, plus Zwei-Wege-CalDAV-Synchronisierung über das Connector-Framework.
- Teilnehmer mit RSVP pro Person, Terminstatus, Farben, Ort und Erinnerungen pro Termin.
- Klicke auf leeren Raum zum Erstellen, ziehe einen Chip zum Verschieben, Mehrfachauswahl für das Verschieben in Massen.
- Zeitzonenbewusste Termine und mehrtägige Bänder, die sich sauber über die Woche legen.
- Gerankte Terminsuche mit Tastaturnavigation, und die Dashboard-Widgets „Heutige Agenda" / „Woche voraus".

## Eine Ansicht, kein separater Speicher

Der Kalender hält keine eigene private Liste von Terminen. Er liest [Objekte](/de/concepts/objects/), die eine Datums-[Eigenschaft](/de/concepts/objects/) haben, und legt sie in der Zeit aus. Erstelle eine Aufgabe in der [Datenbank](/de/apps/database/) mit einem Fälligkeitsdatum und sie erscheint automatisch im Kalender — es ist dasselbe Objekt.

## Mit Zeit arbeiten

- Sieh deine Woche oder deinen Monat auf einen Blick.
- Bewege ein Objekt in der Zeit, indem du es ziehst; die Änderung wird direkt zurück ins Objekt geschrieben.
- Springe von einem Kalendereintrag zum Objekt selbst, um Details hinzuzufügen.

## Verwandt

- [Datenbank](/de/apps/database/) — datierte Objekte erstellen, die hier auftauchen
- [Objekte](/de/concepts/objects/) — die Datumseigenschaften, die der Kalender liest
